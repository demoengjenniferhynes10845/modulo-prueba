# 📝 Gestor de Tareas CLI

Aplicación de consola simple para gestionar tareas con prioridad (Alta, Media, Baja).

## Funciones

- Agregar tareas
- Ver tareas
- Marcar como completadas
- Eliminar tareas
- Guardado en CSV

## Requisitos

- Python 3.7+
- Librería `questionary`

Instalación:

```bash
pip install -r requirements.txt
```

Ejecutar:

```bash
python gestor.py
```
